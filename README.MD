## 🇧🇷 Versão em Português

### 🎯 O Projeto
O objetivo deste projeto foi construir uma vitrine digital que refletisse minha maturidade técnica. Optei por não utilizar frameworks pesados, focando no domínio do **Vanilla JavaScript** para garantir o carregamento mais rápido possível e total controle sobre a experiência do usuário.

### 🛠️ Stack Técnica
* **Frontend:** HTML5, CSS3 (Modern Grid/Flexbox), Vanilla JavaScript.
* **Integrações:** WhatsApp API para contato direto.
* **Metodologias:** Scrum, Clean Code, Comunicação Não Violenta (CNV) na arquitetura de decisões.
* **Hospedagem:** Hostinger.

### ✨ Diferenciais Técnicos
* **Zero Dependencies:** Sem bibliotecas externas desnecessárias.
* **Responsive Architecture:** Layout fluido projetado para breakpoints específicos (< 900px).
* **Performance:** Nota máxima em Core Web Vitals (LCP, FID, CLS).

### 🎯 The Project
This portfolio was built to showcase my technical maturity as a **Full Cycle Developer**. I chose to use **Vanilla JavaScript** to demonstrate core language mastery, ensuring peak performance and total UI control.

* **Frontend:** HTML5, CSS3, Vanilla JavaScript.
* **Integrations:** WhatsApp API for seamless communication.

* **Responsive Logic:** Custom grid logic with specific breakpoint handling.
* **UX/UI Focus:** Designed for clarity, empathy, and conversion.
